Setup/Startup:
Extract the files from cellsData.rar
On the Ipython Notebook, change the basedir (third cell, line 1) to correspond to where the cellsData.rar file has been extracted to.

The following oython libraries are necessary for the program:
  - Scikit-learn
  - Scipy.misc
  - Scikit-image
  - Matplotlib.pyplot (only necessary for plotting graphs)
  - Pandas
  - Numpy
  - Glob
